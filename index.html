Scratch Contributors

About us

We are a group of people who have contributed and still are contributing to Scratch, an educational block-based programming language for children in many ways. We are:

  • Scratch Design Studio (SDS) Curators
  • Scratch Design Studio (SDS) Translators
  • Scratch Wiki Contributors (of all Scratch Wikis)
  • Former Moderators (who used to moderate 1.x)
  • Scratch Translators
  • Scratch Welcoming Committee Curators
  • Scratch Welcoming Committee Translators
  • Propose Projects to be Featured Curators
  • Propose Projects to be Featured Translators
  • Propose Studios to be Featured Translators
  • Front Page Curator Applications Curators
  • Front Page Curator Help Studio Curators
  • Forum Helpers
  • Users who provided Scratch with technology help
  • Users whose names are in the Credits page
  • Scratch Camp Counsellors
  • Scratch Camp Translators
  • Front Page Curator Translators
  • Users who have had pull requests merged into Scratch's GitHub page

About our members

All of our members have contributed to Scratch in different ways! You can read about them here.

How you can join us

If you are a Scratch Contributor and want to join us, then you can either open an issue in our GitHub repository or leave a comment in our studio on Scratch.
